Google Drive — Synchronizacja

⚙️ Konfiguracja (jednorazowa)

Aby synchronizować dane z Google Drive, potrzebujesz klucza API. Poniżej instrukcja krok po kroku — to jednorazowe ustawienie.

1
Wejdź na console.cloud.google.com i zaloguj się kontem Google.
2
Utwórz nowy projekt (np. budowa-kosztorys), następnie w menu wybierz APIs & Services → Library i włącz Google Drive API.
3
Przejdź do APIs & Services → OAuth consent screen → wybierz External → wypełnij nazwę aplikacji → dodaj swój email jako Test user.
4
Przejdź do APIs & Services → Credentials → Create Credentials → OAuth client ID → typ: Web application. W polu Authorized JavaScript origins wpisz adres tej strony (np. https://twoja-strona.pages.dev lub http://localhost jeśli lokalnie).
